# KinderWorld Privacy Policy

This README file contains the Privacy Policy for the KinderWorld application. It explains how user data is handled, what technical data is collected, the use of third-party services, security measures, and contact information.

## 1. Collecte de données

* KinderWorld does not collect any personally identifiable information (name, email, phone number) from users.
* No sensitive or location data is collected without consent.

## 2. Données techniques anonymes

* Technical anonymous data may be collected to improve the application and fix errors.
* These data do not allow personal identification of users.

## 3. Services tiers utilisés

* The application may use third-party libraries or tools (such as Lottie or ExoPlayer) without collecting personal information.
* Personal data is never shared for commercial purposes.

## 4. Enfants

* KinderWorld is intended for children.
* No personal information of children is collected without parental consent.

## 5. Sécurité

* Any anonymous data used is stored securely.
* We are committed to protecting the confidentiality and integrity of this data.
